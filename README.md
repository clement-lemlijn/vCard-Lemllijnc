# vCard - Portfolio Personnel de Clément Lemlijn

Ce projet est un **portfolio personnel** entièrement responsive, conçu pour présenter mes compétences, projets et contact professionnel. Il est développé avec **HTML, CSS et JavaScript**.

## À propos

Le portfolio inclut :

- Présentation personnelle (bio, compétences, parcours)
- Liste de projets avec liens vers les démos et le code source
- Section contact avec formulaire fonctionnel
- Version responsive pour tous types d’écrans (mobile, tablette, desktop)
